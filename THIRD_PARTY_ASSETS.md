## Public Repo Warning

- Sketchfab's standard license page says a standard license does not allow distributing the asset in a way that lets others use or access it as a stand-alone file.
- A public GitHub repo that includes `.glb` files directly may count as stand-alone redistribution.
- Because of that, `public/models/robot_character.glb` and `public/models/smartphone.glb` are the biggest legal risk in this repository as currently packaged.
